The nav element contains links so that the user may either navigate through the web document or any other website location provided.

Typically these navigation element links are listed across the page to navigate within the webpage but does not have to be. nav elements could also be located in an aside element listed and placed off to the side, away from the main content but easily accessible.


The hgroup element is depreciated in HTML5 and is its usage is no longer in style although it is still implemented in most browsers and is unlikely to fully go away.

The hgroup element is merely meant to organize headers and sub headers (also called a heading) defined by elements ranging from h1, the biggest and most important to h6, the smallest and lest important.
An hgroup can be used to isolate a group of headers say for giving it a particular style with an ID or Class, but Again, the element is generally not used.
