The figure figcaption elements are semantic elements necessary for adding graphics and figures to webpages.

These graphics and figures typically provide a visual representation of information in the textual content and referenced by the text. Often you see such images in tutorials and text books in which the author directs the reader to a specific figure.

The progress element represents progress of and object or task.the first element is a header or header group element followed by the section element

The progress element has two attributes; value and max. The value attribute lets you specify the current value or position on the progress element at a specific point in time. The max attribute tells the browser what the maximum possible value is for the element.

The two supported types of progress tasks are determinate and indeterminate. Here we hard-coded values which would be considered determinate because we knew the information we wanted. Examples like this would be downloading a file where you knew the exact size or display the progress of a fundraiser. A indeterminate proggress task is usually assosiated by a CSS psoudo-class that may change how the progress bar is displayed.

Usually the value attribute updates dynamically using JavaScript.
