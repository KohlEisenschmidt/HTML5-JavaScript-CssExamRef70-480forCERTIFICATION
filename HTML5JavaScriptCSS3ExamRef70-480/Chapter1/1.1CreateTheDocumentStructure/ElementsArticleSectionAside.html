The article element contains a distinct and complete entry like a magazine article or blog post.

Each article can have sub-articles but it must be part of the root article as the root article is wholly contained in itself.

Generally within the article element the first element is a header or header group element followed by the section element

The article element subdivides pages into sections, purposefully breaking up the page into more granular details.

For instance a article element would contain zero or more section elements to denote the different content sections within the article. Just like an article element the first element within a section element is typically a header or header group.


The aside elements defines content that does not fall within the main flow or main content of the current page - for example, a sidebar, a note, an alert, or an advertizement.

The aside element does not place itself automatically to any particular side of the webpage; it merely serves as a way to semantically define a section or text or graphics as an aside. later we will see how to position an aside using styles.
